These are the basic steps to run the files
1.First get your alchemyapi key from alchemyapi website.
2. run python alchemyapi.py <your_key>
3.It will generate api_key.text with the api key
4.Then run python recipe.py "search_string" <count>
5.It will find all the tweets with this key word and does sentiment analysis on it and percentage of positive,negative and neutral tweets and find the most positive tweet and most negative tweet.
Also it is storing all the tweets in the file tweet.txt
6.then run R-- something something to produce visualizations in the form of graphs to show the results of sentimental analysis of tweets scraped.
7.File Rplots.pdf will be formed depicting all the graphs formed from mathematical analysis of the scores calculated depicting histograms and graphs.
